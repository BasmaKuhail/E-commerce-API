Project Title: E-Commerce API

Description:
A Spring Boot backend for managing products, categories, brands, tags, images, variants, and reviews. Supports CRUD operations, filtering, and pagination.

Features:
Product CRUD (Create, Read, Update, Delete)
...

Tech Stack:
Java 17
Spring Boot
Spring Data JPA (Hibernate)
MySQL Database
Maven

Project Structure:
..

API Endpoints:
GET /products – get all products
GET /product/{productId} – get product by id
POST /addProduct – create product
PUT /updateProduct – update product
DELETE /delete/{productId} – delete product
...
